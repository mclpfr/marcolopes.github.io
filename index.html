🚗 MLOps Road Accidents - Projet de fin de formation

Classification multiclasse de la gravité des accidents de route (mineur, grave, fatal) avec un pipeline MLOps complet.


Technologies utilisées : Python, Streamlit, MLflow, Airflow, DVC, BentoML, Scikit-Learn


Défis techniques : Dataset déséquilibré, problème de classification multiclasse, déploiement production-ready avec monitoring de dérive des données


Certification : Projet validé dans le cadre du diplôme MLOps DataScientest - F1-score optimisé pour gérer le déséquilibre des classes
